Privacy Policy

Last updated: 22nd February 2025

Overview

LocalBot is committed to protecting your privacy. This policy explains our approach to data handling and privacy protection, emphasizing our no-data-collection stance.

Data Non-Collection Policy

LocalBot operates on a strict no-data-collection principle. We do not collect, store, or process any personal information.

  • No personal identifiers or user data
  • No message content or interaction history
  • No technical information or metadata
  • No cookies or tracking mechanisms

Data Processing

Temporary Processing

  • Real-time command processing
  • No persistent storage
  • Memory cleared on restart

API Interactions

  • Anonymous API requests
  • No user tracking
  • Temporary results only

Security Measures

Our security approach includes:

  • No database implementation
  • Memory-only operations
  • Regular security updates
  • Minimal permission requirements

Policy Updates

We may update this policy to reflect changes in our practices or for other operational, legal, or regulatory reasons.
